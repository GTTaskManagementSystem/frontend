### Installation

1. Clone the repository

```bash
  git clone https://github.com/goteam-ti/frontend.git
  cd frontend
```

2. Install the dependencies

```bash
  yarn install 
```

3. Copy the example env file and make the required configuration changes in the .env file

```bash
  cp .env.example .env
```
